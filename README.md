## Приложение для тестирования
Работает с вопросами из файла csv оформленного по такому примеру: 
```
1;Что такое HTTP?;Протокол передачи гипертекста;Протокол файлов;Протокол безопасности
2;Какой порт по умолчанию у HTTPS?;443;80;8080;22
3;Как выйти из VIM?;format C:\ enter;esc : wq;esc : q!;rm -rf /
```
Где первое значение в строке это номер правильного ответа, второе - сам вопрос, последующие - варианты ответов.
